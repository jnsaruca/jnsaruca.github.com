Why we use the Normalize CSS Script

Normalize.css helps normalize styles of HTML elements. If there are incosistencies, it helps correct bugs and it improves usability. It allows defaults that are useful in different browsers.

What is Semantic Markup

Semantic Markup is structure in HTML that helps interpret the semantic meaning of the content. It helps accurately depict the function of the element.
