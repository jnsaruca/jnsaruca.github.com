Inheritance in CSS

The inheritance in CSS specifies the property into a value from its parent element. It controles what happens when there is no value placed in a property and it sets a default.
Examples:
Color property -> p{color: green;}
Font Family property -> p{font-family: 'Times New Roman';}

Specificity in CSS

Specifity applies to when one element has multiple declarations. If there are multiple declations,the last one applies to the element. Specifity puts weight to CSS declaration through the number of selector type.
A Combinator Selector selects elements based on some specific relationship between them.
Multiple Selectors share the same declarations that allows application of styles to multiple elements.

ID and Class Attributes

The ID Attributes is a unique identifier. It is used to specify a certain document to assign a certain task for a unique element. It is written using a # symbol before the id.
The class attribute is used for one or more class names. It can be used for any elements in HTML. It can also be used to do certain tasks for CSS and Javascript with a specific class name.
